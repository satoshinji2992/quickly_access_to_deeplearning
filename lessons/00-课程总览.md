# 课程总览

这套教程不是按名词排列，而是按问题推进。

## Block 1：基础知识

目标：知道模型训练到底在做什么。

任务顺序：

1. [线性回归](../exercises/block_01_basics/task_00_linear_regression/README.md)
2. [圆形分类 MLP](../exercises/block_01_basics/task_01_circle_classifier/README.md)
3. [小型深度学习库](../exercises/block_01_basics/task_02_mini_dl_lib/README.md)
4. [MNIST MLP](../exercises/block_01_basics/task_03_mnist_mlp/README.md)

这块结束后，你应该能说清楚 forward、loss、backward、optimizer、batch 和 epoch 是怎么连起来的。

## Block 2：ResNet

目标：用自己的 NumPy 小库训练图像分类模型。

任务顺序：

1. [图像数据管线](../exercises/block_02_resnet/task_10_image_data_pipeline/README.md)
2. [Conv2D 与 im2col](../exercises/block_02_resnet/task_11_conv2d_im2col/README.md)
3. [池化与 BatchNorm](../exercises/block_02_resnet/task_12_pooling_and_bn/README.md)
4. [残差块](../exercises/block_02_resnet/task_13_residual_block/README.md)
5. [NumPy ResNet 训练](../exercises/block_02_resnet/task_14_numpy_resnet_train/README.md)
6. [实验报告](../exercises/block_02_resnet/task_15_resnet_report/README.md)

这块会比较硬。不要跳过梯度检查，否则后面调模型会很痛苦。

## Block 3：Transformer 与 MiniMind

目标：用 PyTorch 搭出一个小型 decoder-only 语言模型。

任务顺序：

1. [Transformer 理论](../exercises/block_03_transformer/task_20_transformer_theory/README.md)
2. [交叉注意力](../exercises/block_03_transformer/task_21_cross_attention/README.md)
3. [正弦位置编码](../exercises/block_03_transformer/task_22_sinusoidal_position/README.md)
4. [RoPE 位置编码](../exercises/block_03_transformer/task_23_rope_position/README.md)
5. [多头注意力](../exercises/block_03_transformer/task_24_multi_head_attention/README.md)
6. [前馈网络](../exercises/block_03_transformer/task_25_feedforward/README.md)
7. [Embedding 与 LM Head](../exercises/block_03_transformer/task_26_embedding_and_lm_head/README.md)
8. [多层叠加](../exercises/block_03_transformer/task_27_stack_transformer_blocks/README.md)
9. [MiniMind Core](../exercises/block_03_transformer/task_28_minimind_core/README.md)
10. [具体任务训练](../exercises/block_03_transformer/task_29_minimind_training/README.md)

这一块对齐开源 MiniMind 的核心思路：RMSNorm、RoPE、SwiGLU、GQA 和 decoder-only 训练。
