# LoRA

LoRA 是一种参数高效微调方法.

全量微调会更新模型里大量参数. LoRA 不这么做.

它冻结原模型权重, 只在某些线性层旁边加一小组低秩矩阵:

$$
W' = W + BA
$$

其中 $A$ 和 $B$ 的秩很小.

## 为什么常和 SFT 一起出现?

SFT 数据量通常比预训练小很多.

如果每次都全量微调, 显存和存储压力都很大.

LoRA 只训练少量新增参数, 更适合个人设备和小规模实验.

## 代价

LoRA 不是免费午餐.

rank 太小, 可能学不动.

rank 太大, 又会接近全量微调的成本.

所以它常常需要调 `r`、`alpha`、目标层和学习率.
