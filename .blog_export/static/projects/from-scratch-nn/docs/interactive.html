每一章的统一模板

  • 问题：这章到底在解决什么。
  • 直觉：先让读者理解为什么需要它。
  • 数学式：只保留关键公式，不堆导数墙。
  • 最小代码：能直接运行，变量名与公式统一。
  • 结果图：loss 曲线、决策边界或生成样例。
  • 练习：读者知道自己该动手写什么。

交互 02

任务二：为什么需要非线性层

这个交互不是完整训练器，而是一个“前向传播可视化”。你可以拖动平面上的点，观察隐藏层如何把二维输入变成对“圆内 / 圆外”的判断。

0.000 输出 logit
0.000 圆内概率
外部 模型判断
外部 真实标签
这里用一组固定权重模拟“隐藏层学会了半平面与折线边界”的效果。隐藏单元越多，近似圆形的能力越强，这正是教程里从线性模型过渡到 MLP 的关键直觉。

输入 → 隐藏层

你希望读者在这页学会什么

  • 线性分类器只能画一条线，无法直接分开圆形边界。
  • 隐藏层中的多个神经元可以组合出复杂边界。
  • 最终输出层给出的是一个 logit，再经过 Sigmoid 变成概率。
  • 这页负责建立直觉，下一步再进入损失函数和反向传播。
