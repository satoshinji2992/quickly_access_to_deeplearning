## step0.2 预备知识

### 完成任务一，你所需要了解的部分numpy用法：
```python
import numpy as np

# 创建随机数组（服从标准正态分布）
x = np.random.randn(2) #括号内的是数组的形状
y = np.random.randn(11,45,14) #可以创建多维张量


# 基本数学运算（逐元素计算）
z = x + y  # 逐元素相加，要求加与被加张量形状相同
z = x * 2  # 数乘
z = x * y  # 逐元素相乘（即哈达玛积），要求乘与被乘张量形状相同
z = x ** 2  # 平方
z = x @ y  # 向量点积或矩阵乘法


# 统计函数
mean_x = np.mean(x)  # 平均值
sum_x = np.sum(x)    # 求和
```
### 如何创建和加载数据（panda csv相关的用法）：
```python
import pandas as pd

# 方法1：直接创建DataFrame
data = pd.DataFrame({
    'x': [1, 2, 3, 4],
    'y': [2, 4, 6, 8]
})

# 方法2：从CSV文件读取（推荐）
data = pd.read_csv('Salary_Data.csv')

# 查看数据基本信息
print(data.head())      # 前5行
print(data.shape)       # 数据形状
print(data.columns)     # 列名

# 提取x和y
x = data['x_column'].values  # 替换为实际的列名
y = data['y_column'].values
```

### hint2 超参数调优

调整学习率：从0.05, 0.01, 0.001等不同值尝试
增加迭代次数：有些复杂数据需要更多训练
数据标准化：如果x和y尺度差异大，可以先标准化
```python
# 数据标准化
x_normalized = (x - np.mean(x)) / np.std(x)
y_normalized = (y - np.mean(y)) / np.std(y)
```
