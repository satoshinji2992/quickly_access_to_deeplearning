## 这套教程在讲什么

主线一直很简单：

`一条直线 -> 非线性分类 -> 手搓 MLP -> 整理小型深度学习库 -> 图像里的 ResNet -> 文本里的 Transformer`

如果你能把前四步真的自己写下来，后面看 PyTorch 里的大部分模型时，心里都会更踏实。

## 每一章解决什么问题

### [02-从直线到梯度下降.md](/Users/shenqi/Documents/code/深度学习入门/lessons/02-从直线到梯度下降.md)

从最简单的线性回归出发，搞明白模型、参数、损失函数、梯度和学习率到底是什么。

### [03-圆形分类与MLP.md](/Users/shenqi/Documents/code/深度学习入门/lessons/03-圆形分类与MLP.md)

解释为什么直线分不开圆，引出激活函数、交叉熵、多层感知机和反向传播。

### [04-完善你的小型深度学习库.md](/Users/shenqi/Documents/code/深度学习入门/lessons/04-完善你的小型深度学习库.md)

把前面零散的代码整理成可复用的层、激活函数、优化器、归一化和正则化模块。

### [05-ResNet图像分类.md](/Users/shenqi/Documents/code/深度学习入门/lessons/05-ResNet图像分类.md)

从卷积为什么适合图像开始，一直走到残差块、训练循环和 CIFAR-100 上的简化版 ResNet 实验。

### [06-Transformer与文本生成.md](/Users/shenqi/Documents/code/深度学习入门/lessons/06-Transformer与文本生成.md)

这一部分会把文本生成项目逐步并回主线，目前先给出方向和项目入口。

## 现在最适合怎么学

- 不要一上来直接看 `solutions/`
- 每学完一章，至少把对应的最小代码自己敲一遍
- 训练时盯住 loss 和 accuracy，不要只管代码能不能跑
- 真卡住了再看参考实现
