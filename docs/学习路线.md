## 推荐顺序

### 第一阶段：先把训练过程看懂

1. [课程主线](/Users/shenqi/Documents/code/深度学习入门/lessons/01-课程主线.md)
2. [从直线到梯度下降](/Users/shenqi/Documents/code/深度学习入门/lessons/02-从直线到梯度下降.md)
3. [任务一实践引导](/Users/shenqi/Documents/code/深度学习入门/lessons/02-任务一实践引导.md)
4. [任务一练习代码](/Users/shenqi/Documents/code/深度学习入门/exercises/task_00_linear_regression/mission_0.py)

这一阶段的目标是理解：

- 模型是什么
- 参数是什么
- 损失函数怎么衡量好坏
- 梯度下降为什么能更新参数

### 第二阶段：进入非线性分类

1. 阅读[圆形分类与MLP](/Users/shenqi/Documents/code/深度学习入门/lessons/03-圆形分类与MLP.md)
2. 阅读[任务二实践引导](/Users/shenqi/Documents/code/深度学习入门/lessons/03-任务二实践引导.md)
3. 运行[任务二练习代码](/Users/shenqi/Documents/code/深度学习入门/exercises/task_01_circle_classifier/mission_1.py)
4. 阅读[完善你的小型深度学习库](/Users/shenqi/Documents/code/深度学习入门/lessons/04-完善你的小型深度学习库.md)

这一阶段的目标是理解：

- 为什么线性模型分不开圆
- 为什么需要激活函数
- MLP 的前向传播和反向传播在做什么
- 如何开始搭自己的小型深度学习库

### 第三阶段：进入图像分类

继续阅读[ResNet图像分类](/Users/shenqi/Documents/code/深度学习入门/lessons/05-ResNet图像分类.md)。

这一阶段的目标是理解：

- 卷积层为什么适合图像
- 残差连接解决了什么问题
- 一个标准图像分类训练流程长什么样

### 第四阶段：进入文本生成

先读[Transformer与文本生成](/Users/shenqi/Documents/code/深度学习入门/lessons/06-Transformer与文本生成.md)，再参考[Transformer 文本生成项目](/Users/shenqi/Documents/code/深度学习入门/exercises/task_04_transformer_textgen/README.md)。

这一阶段的目标是理解：

- Self-Attention 的基本作用
- Transformer 的最小训练流程
- 文本生成项目的组织方式
